[size=4][b]What It Does[/b][/size]

QLP is an [url=https://www.nexusmods.com/skyrimspecialedition/mods/6194]Mod Organizer 2 (MO2)[/url]  "Quick Link" plugin for [url=https://phostwood.github.io/crash-analyzer/skyrim.html]Phostwood's Crash Log Analyzer[/url]. From crash to analysis in [b]zero clicks[/b] — QLP now detects new crash logs when Skyrim closes/crashes and can automatically upload them to [b]Sovnkrasch[/b] (our dedicated cloud storage for Skyrim crash logs) and open the analyzer in your browser, preloaded and ready to review within seconds.

Fully configurable: enable Local-Only mode to skip the upload entirely, adjust auto-trigger behavior, set crash log paths and filename patterns, and more — all from MO2's settings panel. Or trigger it manually anytime from the Tools menu.

Uploaded logs get a permanent* shareable link — useful when asking for help, sharing an interesting crash as an example, or contributing to the broader pool of researchable Skyrim crash data.

[list]
[*][b]Auto-Finds Your Log[/b] — Locates your most recent crash log automatically. Supports the preferred CrashLoggerSSE out of the box; paths and filename patterns are configurable.
[*][b]Zero-Click Analysis[/b] — Detects new crash logs when Skyrim closes and can automatically open the analyzer. No need to open the Tools menu after a crash. Configurable: prompt each time, analyze silently, or disable entirely.
[*][b]Local-Only Mode[/b] — Skip the upload entirely. The analyzer opens in your browser and automatically copies your crash log into your clipboard, letting you paste your log in manually.
[*][b]Automatic Fallback[/b] — If an upload fails, the plugin falls back to local-only behavior automatically so you're never left without a result.
[*][b]Configurable[/b] — Adjust auto-trigger behavior, crash log folder path, filename patterns, and more from MO2's settings panel. Defaults cover most setups without any configuration.
[*][b]Shareable Links[/b] — Uploaded logs get a permanent* link you can share with others — no expiration, no fighting Pastebin size limits.
[/list]
[i]* "Permanent" in theory, but I'm trying to keep this all under free tier storage, so it's possible the old logs may eventually get aged out[/i]

[size=4][b]Powered by Sovnkrasch[/b][/size]

[img]https://phostwood.github.io/crash-analyzer/images/Sovnkrasch-339x339.png[/img]

When you choose to upload, QLP sends your log to [i]Sovnkrasch[/i] — a cloud storage service built specifically for Skyrim crash logs, running on Cloudflare Workers and R2. The name came from Kyler45: [i]sovn[/i] means "sleep" in Norwegian, [i]krasch[/i] means "crash." A crash log's final resting place.

[b]What makes it different:[/b]

[list]
[*][b]Privacy-first[/b] — No personal information collected; Windows usernames are automatically redacted from file paths. [url=https://phostwood.github.io/crash-analyzer/PrivacyPolicy.html]Full privacy policy.[/url]
[*][b]Skyrim-only[/b] — Validates uploads to ensure only actual Skyrim crash logs are accepted.
[*][b]Smart deduplication[/b] — Upload the same log twice and you get the same link back.
[*][b]Handles big logs[/b] — Accepts up to 2 MB compressed. A 35 MB uncompressed log has been successfully uploaded.**
[/list]

[i]** For comparison: pastebin.com caps at 512 KB uncompressed. Sovnkrasch accepts up to 2 MB compressed — so even monster logs fit.[/i]

[size=4][b]Installation & Setup[/b][/size]

All you need is [url=https://www.nexusmods.com/skyrimspecialedition/mods/6194]MO2 [/url]and a crash log with the default filename format — e.g. [font=Courier New]crash-2026-02-16-04-41-28.log[/font]. [i] [/i][url=https://www.nexusmods.com/skyrimspecialedition/mods/59818]Crash Logger SSE[/url][b][i]﻿ highly recommended[/i][/b][i]. [/i]Takes about five minutes.

[b]1. Download the Plugin[/b]
Grab [font=Courier New]MO2Plugin-PhostwoodQLP.py[/font] from Files tab (or [url=https://github.com/Phostwood/crash-analyzer/blob/main/MO2Plugin-PhostwoodQLP.py]GitHub[/url]).

[b]2. Drop It in Your MO2 Plugins Folder[/b]
Move the file into your MO2 [font=Courier New]plugins[/font] directory, e.g.:
[code]C:\Modding\MO2\plugins[/code]

[b]3. Start (or Restart) MO2[/b]
The plugin loads automatically. If MO2 was already running, close and reopen it.

[b]4. Run It[/b]
Go to [font=Courier New]Tools → Tool Plugins → "QLP: Upload Crash Log and Open Analyzer"[/font].

[b]5. Optional: Configure It[/b]
Adjust your crash log folder path and filename pattern at [font=Courier New]Tools → Settings → Plugins → "QLP: Upload Crash Log and Open Analyzer"[/font].

Default configurations:
analyzer_url:  [code]https://phostwood.github.io/crash-analyzer/skyrim.html[/code]
crash_log_dir: (leave blank to default to user's SKSE directory)
crash_log_globs: [code]crash-*.log, Crash_*.txt[/code]
last_processed_log_time: (auto-managed; clear to force re-trigger on next exit)
local_only_mode: [code]false[/code]
on_game_exit_behavior: [code]prompt[/code]
worker_url: [code]https://skyrim-crashlog-worker.phostwood.workers.dev/upload
[/code]
