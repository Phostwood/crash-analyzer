# Phostwood's Crash Analyzers:

With over 500 hours of development, this free, open source, advanced analyzer examines Nolvus and general Skyrim crash logs to help diagnose and fix 80-90% of crashes, providing well-researched troubleshooting steps and links. It's currently helping over 100 different Skyrim modders per day, and stands apart from other analyzers with its advanced diagnoses and frequent updates.

Both share 98% of the same code, and both have graduated from beta. Usage requires an internet connection, is hosted for free by GitHub Pages:
